Module 5: City Connectivity & Resource Planning

This module focuses on analyzing city networks to ensure connectivity and optimize the reachability of critical resources. Using shortest path trees, we aim to plan efficient routes and resource allocations across the city.

Algorithm Used:

Shortest Path Trees (Dijkstra/Prim), Connectivity checks, and Dynamic Programming were applied to optimize reachability and resource distribution. (Optional: BFS/DFS can be used to explore alternative routes and improve connectivity planning.)
