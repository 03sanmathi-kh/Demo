Traffic Signal Optimization Using EV Priority

Problem Statement

Emergency electric vehicles such as ambulances, fire trucks, and police vehicles experience delays due to fixed-time traffic signals. These delays increase response time and can result in loss of life.

Solution

  • Detect approaching emergency EVs in real time
  • Dynamically adjust traffic signals
  • Create a green corridor for emergency vehicles
  • Minimize disruption to normal traffic flow

Data Structures Used

  • Graph → Road intersections and connectivity
  • Queue → Vehicles waiting at signals
  • Priority Queue → Emergency vehicle prioritization

Algorithms Used

  • Dijkstra’s Algorithm → Shortest emergency route
  • BFS → Signal update propagation

Algorithm Efficiency

  • Dijkstra’s Algorithm: O(E + V log V) — efficient for dynamic route calculations.
  • BFS: O(V + E) — fast propagation of congestion updates across intersections.

Step-by-Step Process

  • Emergency EV detected via sensors
  • Road network modeled as a graph
  • Dijkstra finds fastest route
  • BFS updates all upcoming signals
  • Signals turn green dynamically
  • Normal traffic resumes after EV passes
